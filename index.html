Sale Transaction Details
All fields are required. Triggers enforce validity.

EstateDesk › About

About This Project

CS241 · DBMS Project · Deliverable 4 · IIIT Guwahati

Project Overview
real_estate_db · MySQL 8 · Java JDBC + Spring Boot backend · HTML/CSS/JS frontend

EstateDesk is the Deliverable 4 interface for the CS241 Real Estate DBMS project. It provides a browser-based portal for real estate agents to view sales reports, rental reports, and record new transactions — all backed by a Spring Boot REST API connected to the real_estate_db MySQL database via JDBC.

Team Contributions
Deliverable 4 — Interface for real estate office and agents
Database Schema (Key Tables)
All 9 tables in real_estate_db used in this interface

Property(property_id, owner_id, state, city, locality, house_number, unit_type)

Sale_Transactions(txn_id, listing_id, agent_id, deal_date, final_price)
Rental_Transactions(txn_id, listing_id, agent_id, rent_start, rent_end, deal_date)
